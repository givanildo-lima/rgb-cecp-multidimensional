KTH-TIPS2-b image database

Please go to the website

	http://www.nada.kth.se/cvap/databases/kth-tips

for documentation before using this database.

The eleven materials are placed in separate directories

    aluminium_foil
    brown_bread
    corduroy
    cork
    cotton
    cracker
    lettuce_leaf
    linen
    white_bread
    wood
    wool

each containing the four samples in separate directories

    sample_a
    sample_b
    sample_c
    sample_d

in which the images are clearly labelled with respect to scale and
image label 1-12 (see the documentation for what the labels 1-12
mean).
